# Modelo de Previsão de Vendas

Projeto em Python para previsão de vendas com base no valor investido em marketing, utilizando regressão linear. A aplicação permite analisar dados, treinar um modelo preditivo e testar estimativas diretamente pelo terminal, além de visualizar os resultados em gráfico.

## Objetivo

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de análise de dados e machine learning de forma prática, prevendo a quantidade de vendas a partir do investimento realizado.

## Tecnologias utilizadas

- Python
- Pandas
- Scikit-learn
- Matplotlib

## Funcionalidades

- Leitura de dados a partir de arquivo CSV
- Treinamento de modelo de regressão linear
- Previsão de vendas com base em um valor informado
- Exibição de gráfico com dados reais e previsão
- Interação simples via terminal
